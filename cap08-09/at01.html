O Caminho para a Iluminação

Há uma necessidade contínua de mostrar o poder das tecnologias web. Este projeto propõe-se a explorar ao máximo os limites das folhas de estilo em cascata (CSS).

Sobre o que é isso?

O HTML desta página é estático. Apenas o arquivo CSS é substituído para que o layout mude completamente. Isso prova a separação perfeita entre o conteúdo (HTML) e a apresentação (CSS).

Participação

O design forte visual sempre foi o foco. Use a sua imaginação, utilize Flexbox, CSS Grid, variáveis e crie algo impressionante.
