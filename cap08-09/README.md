# Projeto CSS Zen Garden - Capítulos 8 e 9

## Autor

Cauê Nairan Ferreira

## Tema

Cyberpunk / Neon Dark Mode

## Design de Referência

Layouts modernos com inspiração "Neon Cyberpunk" e interfaces de terminais futuristas.

## Descrição

Este projeto recria o visual do CSS Zen Garden focando em um layout escuro e moderno. O layout utiliza fortemente o **CSS Grid** para a macroestrutura da página, separando o conteúdo principal da barra lateral (sidebar) em resoluções maiores. O **Flexbox** foi empregado para micro-layouts, como o alinhamento dos menus de navegação, centralização de textos e ícones.

## Tecnologias e Recursos Utilizados

- **CSS Variables (:root):** Para fácil manutenção da paleta de cores e tipografia.
- **CSS Grid Layout:** Estruturação principal do site (Main vs Sidebar).
- **Flexbox:** Alinhamento de elementos internos (listas, links e rodapé).
- **Tipografia:** Uso da fonte _Fira Code_ (Google Fonts) para reforçar o aspecto "tech".
- **Media Queries:** Responsividade completa do Mobile ao Desktop.
- **Efeitos de Hover e Transições:** Glow effects (sombras neon) e movimentação em links.
- **Desafio Bônus:** O tema já é um "Dark Mode" nativo com uso de animações suaves no cabeçalho.
