3. Glossário Técnico

Node
Um ponto de interseção ou conexão dentro de uma rede.
No contexto de HTML/DOM, qualquer elemento da árvore do documento (elemento, texto, atributo).
Bug
Um erro ou falha em um programa de computador.
Comportamento inesperado de um software que produz um resultado incorreto.
Frontend
Client-side
A parte da aplicação web que executa diretamente no navegador do usuário.
Backend
Server-side
A parte da aplicação que executa no servidor, gerenciando bancos de dados e regras de negócio.
HTML
HyperText Markup Language, linguagem padrão para criar páginas web.
CSS
Cascading Style Sheets, usado para estilizar a apresentação de um documento HTML.
API
Application Programming Interface, um conjunto de rotinas e padrões de programação.
DOM
Document Object Model, interface de programação para documentos HTML e XML.
