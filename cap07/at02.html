O Início de Tudo

A World Wide Web mudou a forma como a sociedade interage. Inicialmente composta por páginas estáticas, a web servia apenas para partilha de documentos científicos.

"A Web não liga apenas máquinas, ela liga pessoas." - Tim Berners-Lee

A Era Moderna

Hoje, com HTML5, CSS3 e JavaScript, construímos aplicações ricas. O Box Model é fundamental para compreender como os elementos ocupam espaço no ecrã.

A introdução de Web Fonts permitiu uma tipografia muito mais rica, libertando os designers das fontes padrão dos sistemas operativos.
